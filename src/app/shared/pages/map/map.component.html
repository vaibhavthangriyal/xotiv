Flow to follow to use this application
  • 1. Add User Role : This is need to create a user.
  • 2. Add User (Using User Role) : This can be driver or any other role
  • 3. Add Fuel Type : It is needed for the vehicle creation
  • 4. Add Vehicle Type : It is need fot the vehicle creation
  • 5. Add Vehicle : It needs driver (user), fuel type , vehicle type
  • 6. Generate Locations (By clicking on Buttons) : This will assign some predefined lat, lng to that vehicle . All locations are of Delhi NCR region
  • 7. Go to map (by clicking on Maptracker on top left on navbar)
  • 8. Click Track to track its location : This will create a route and will also mark all the locations on map
Map Screen
Map Screen will show google map. And will always be centered on your current location.
On Right side you will see list of all vehicles and thier Details.
Once you Will click track button . It create a route from its first location to its last location and will also show marker on every location with address of that location.
